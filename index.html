Cyberus' GDG 2024 Quizzes

Welcome Challengers, Are you ready to test your coding skills? You are required to solve at least one problem to claim the prize, which includes Cyberus' stickers and T-Shirt.

Rules

  • Challengers can use any programming language of choices.
  • For fairness, Challengers can not use any pre-built libraries or functions that performs complex operations such as sorting , searching, etc. If found using such libraries or functions, challenger will be disqualified.
  • Challengers are required to do a Write-up to claim the prize.
  • Challengers are prohibited to share solutions with other challengers.

How to earn prizes

  • Solve one easy problem -> Stickers.
  • Solve two medium problems -> Cyberus T-Shirt.
  • Solve one hard problem -> Cyberus T-Shirt and Stickers.
  • Prizes DO NOT stack.

Challenge pool

Easy

> Fibonacci Sequences

Hello Challenger, I really need you to create this thing called Golden Ratio for me, but doing it by hand is reaallly difficult. Could you please help me create this using your extraordinary skills and create a golden ratio of 39 iterations for me, you will be rewarded greatly!

Example

  • f(0) -> 0
  • f(3) -> 2 : 1 1 2
  • f(5) -> 5 : 1 1 2 3 5

Answer

The answer is the number of Fibonacci sequences of 39.

The answer is the number of Fibonacci sequences of 1337

> Look and Say

Hey Challenger, Let's play a game and see if you can read out what you see!

Example

5 rounds starting with 1:

  • 1 -> 11 (one 1)
  • 11 -> 21 (two 1)
  • 21 -> 1211 (one 2 one 1)
  • 1211 -> 111221 (one 1 one 2 two 1)
  • 111221 -> 312211 (three 1 two 2 one 1)

Answer

The answer is the length of the total number when starting with 1113322113 and continue the game for 39 rounds.

Hard

> Weird Grimoire

The great mage Frieren found a grimoire from her side quests, the grimoire contains weird symbols that she has never seen before. However, the book has a description that says:

A MAGE THAT IS CAPABLE ENOUGH TO FIND WHAT REGISTER a HOLDS WILL UNLOCK A SECRET MAGIC BEHIND THIS GRIMOIRE.

Challenger, now it's your job is to help Frieren find what is the value register a has to unlock the secret magic.

Notes

  • Every register is 16 bit.
  • The input contain 5 instructions: AND, OR, LSHIFT, RSHIFT and NOT.
  • Use this input file to find the answer.

Example

  • 829 -> x
  • 921 -> y
  • x AND y -> a
  • a OR y -> b
  • b LSHIFT 2 -> c
  • c RSHIFT 2 -> d
  • NOT x -> e

Final value:

  • x = 829
  • y = 921
  • a = 793
  • b = 921
  • c = 3684
  • d = 921
  • e = 64706

Answer

The answer is the value of the register a from the input file provided.
